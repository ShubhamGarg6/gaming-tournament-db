# Online Gaming Tournament Management System

## Description
This project is a database system that manages players, teams, tournaments, matches, and scores.

## Features
- Add Player
- View Players
- Update Player
- Delete Player

## Technologies Used
- SQLite
- Java
- JDBC

## How to Run
1. Add SQLite JDBC driver
2. Run MainApp.java
3. Use menu options
